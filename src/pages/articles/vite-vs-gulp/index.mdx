## âš™ï¸ What is Gulp?

**Gulp** is a JavaScript-based task runner introduced in 2013, designed to automate repetitive workflows like:

- Compiling Sass/Less
- Minifying CSS/JS
- Image optimization
- Watching files for changes
- Live reload during development

Gulp works by chaining plugins in a pipeline using `gulpfile.js`.

Example:

```js
const gulp = require('gulp')
const sass = require('gulp-sass')(require('sass'))
const cleanCSS = require('gulp-clean-css')

gulp.task('styles', () =>
  gulp
    .src('src/scss/**/*.scss')
    .pipe(sass())
    .pipe(cleanCSS())
    .pipe(gulp.dest('dist/css'))
)
```

## âš¡ What is Vite?

**Vite**, created by Evan You (creator of Vue), is a **next-gen frontend tool** designed for blazing-fast development and optimized builds.

Key features:

- Native ES modules support
- Lightning-fast hot module replacement (HMR)
- Out-of-the-box support for TypeScript, Vue, React, Svelte
- Built-in dev server and build pipeline (using Rollup under the hood)

Example config (`vite.config.ts`):

```ts
import { defineConfig } from 'vite'

export default defineConfig({
  root: 'src',
  build: {
    outDir: '../dist',
  },
})
```

### âœ… Vite Pros

- Extremely fast cold starts and rebuilds
- Native support for modern JS frameworks
- Built-in optimizations (code splitting, minification, tree-shaking)
- Easier to configure than Webpack

- Less suited for non-module-based workflows (e.g., multi-page templates with Handlebars)
- Requires adapting if migrating from traditional task runners
- Plugin ecosystem is still growing (compared to Gulp's ecosystem)
